# Brain Boost Kids

A colorful educational game for children built with vanilla HTML, CSS, and JavaScript.

## Included files

- `index.html` — Main game landing page and Missing Numbers experience.
- `style.css` — Bright, playful styling with responsive layout.
- `script.js` — Game engine, random question generation, scoring, stars, sound toggle, and confetti.
- `missing-shapes/index.html` — Separate Missing Shapes game page.
- `missing-shapes/style.css` — Styling for the shapes game.
- `missing-shapes/script.js` — Shape puzzle logic with pattern recognition rounds.
- `assets/icons/` — Folder for child-friendly icon artwork.

## Features

- Home page navigation between Missing Numbers and Missing Shapes
- Multiple Missing Numbers modes: 1–20, 1–50, 1–100, Greater Than, Less Than, Between Numbers
- Difficulty settings: Easy, Medium, Hard
- 10-question rounds with score tracking
- 3-star reward system and level progression
- Animated feedback, hint panel, sound toggle, and restart support
- Mobile responsive layout suitable for future Capacitor Android wrapping
- Child-safe with no ads or login

## Run locally

Open `index.html` directly in a browser or use Live Server in VS Code.

## Notes

- The game uses CSS and vanilla JavaScript only.
- The `missing-shapes` folder contains a second game experience with shape puzzles and bright interaction.
